# Resources for Structured Programming Language (CSE 1102)

## Syllabus Topics
- C Programming Basics
- Control Structures (if-else, loops)
- Arrays (1D, 2D) & Strings
- Functions & Recursion
- Pointers & Memory Management
- Structures & Unions
- File Handling

## Recommended Books
- **The C Programming Language** by Brian Kernighan and Dennis Ritchie (K&R)
- **Teach Yourself C** by Herbert Schildt

## Online Resources
- [CS50's Introduction to Computer Science (Harvard/edX)](https://cs50.harvard.edu/) - Best for concepts.
