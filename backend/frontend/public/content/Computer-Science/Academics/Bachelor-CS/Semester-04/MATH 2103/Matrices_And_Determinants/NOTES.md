# Matrices & Determinants

## Basics
- **Matrix:** Rectangular array of numbers.
- **Determinant:** Scalar value associated with a square matrix.

## Key Concepts
1.  **Rank of Matrix:** Maximum number of linearly independent rows/columns.
2.  **Inverse:** A^-1 = adj(A) / det(A). Only exists if det(A) ≠ 0.
3.  **Eigenvalues & Eigenvectors:**
    *   Equation: (A - λI)X = 0
    *   Roots (λ) are eigenvalues.

## Systems of Linear Equations
- **Consistency:** If Rank(A) = Rank(A|B), system is consistent.
- **Cramer's Rule:** Method to solve system using determinants.
