# Divide and Conquer

## Concept
Divide the problem into smaller sub-problems, solve them, and combine the results.

## Key Algorithms
1.  **Merge Sort:**
    *   Divides array into two halves.
    *   Recursively sorts them.
    *   Merges sorted halves.
    *   Time Complexity: O(n log n).

2.  **Quick Sort:**
    *   Picks a pivot.
    *   Partitions array (smaller elements left, larger right).
    *   Recursively sorts parts.
    *   Best/Avg: O(n log n), Worst: O(n^2).

3.  **Binary Search:**
    *   Finds element in sorted array.
    *   Time Complexity: O(log n).
