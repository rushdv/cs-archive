# Coordinate Geometry (2D & 3D)

## 2D Geometry
- **Straight Lines:** y = mx + c
- **Conic Sections:**
    - **Circle:** x² + y² = r²
    - **Parabola:** y² = 4ax
    - **Ellipse:** x²/a² + y²/b² = 1
    - **Hyperbola:** x²/a² - y²/b² = 1

## 3D Geometry
- **Direction Cosines:** Angles made with X, Y, Z axes.
- **Plane Equation:** ax + by + cz + d = 0
- **Sphere:** (x-a)² + (y-b)² + (z-c)² = r²
