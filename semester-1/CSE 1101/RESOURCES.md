# Resources for Introduction to Computers (CSE 1101)

## Syllabus Topics
- Computer Generations & History
- Number Systems (Binary, Octal, Hex, Conversions)
- Logic Gates & Boolean Algebra
- Basic Computer Architecture (CPU, Memory, I/O)
- Operating System Basics
- Computer Networks & Internet

## Recommended Books
- **Computer Fundamentals** by P.K. Sinha
- **Introduction to Computers** by Peter Norton
